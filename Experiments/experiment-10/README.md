# Experiment 10: CRUD Operations using Node.js, Express.js, and MongoDB

## 📌 Aim
To implement CRUD (Create, Read, Update, Delete) operations using Node.js, Express.js, and MongoDB.

---

## 🧠 Description
This project is a RESTful API that performs basic CRUD operations on student data. It allows users to create, read, update, and delete student records stored in a MongoDB database.

The backend is built using Express.js, and MongoDB is used as the database with Mongoose for schema modeling.

---

## 🛠️ Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman (for API testing)
